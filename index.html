Logika

A klasszikus kétértékű logikában csak igaz, hamis értéket használunk(0 vagy 1). Viszont előfordulhat, hogy szükségünk van 3. értékre is (ismeretlen, vagy nemtudom), mert nem tudjuk egyértelműen eldönteni, hogy valami igaz-e. Az ismeretlen értéke:0.5, vagy 1/2.

Vizsgáljuk meg az alapműveleteket:

1. Negáció (NEM, !):

A !A
0 1
1/2 1/2
1 0

2. Konjunkció (ÉS, &&)

A B A && B
0 0 0
0 1/2 0
0 1 0
1/2 0 0
1/2 1/2 1/2
1/2 1 1/2
1 0 0
1 1/2 1/2
1 1 1

 

 

3. Diszjunkció (VAGY, ||)

A B A || B
0 0 0
0 1/2 1/2
0 1 1
1/2 0 1/2
1/2 1/2 1/2
1/2 1 1
1 0 1
1 1/2 1
1 1 1

4. Implikáció (A-ból következik B, >)

A > B <=> (!A || B)

 

5. Ekvivalencia (A ekvivalens B-vel, =

(3 vonal lenne csak azt nem engedi a gép))

(A = B ) <=> (A > B) && (B > A) <=> (!A || B) && (!B || A)

  

 

Az ekvivalencia műveletét lehet másképp is értelmezni, az úgymond ránézéses módszerrel. Ha az értékek megegyeznek, akkor igaz, ellenben hamis.

A B A = B
0 0 1
0 1/2 0
0 1 0
1/2 0 0
1/2 1/2 1
1/2 1 0
1 0 0
1 1/2 0
1 1 1
